
# Scientific Calculator

This is a Python-based scientific calculator built using Tkinter. It performs a wide range of mathematical operations in a simple and user-friendly interface.

Features include:

-Trigonometric functions such as Sin, Cos, and Tan, with support for degrees and radians.

-Logarithms (ln, log) and their inverse functions.

-Square roots, factorials, powers, and other common mathematical operations.
